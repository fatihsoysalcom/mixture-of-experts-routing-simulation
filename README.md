# Mixture of Experts Routing Simulation

This example simulates a simplified Mixture-of-Experts (MoE) architecture. It demonstrates how a 'router' intelligently directs different types of inputs to specialized 'experts', activating only a subset of the model for each query. This sparse activation is a core principle of MoE, designed to reduce computational inference costs compared to traditional large models.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
